# Healthcare-Analytics-Dashboard-Power-BI-
## Overview
This project presents an interactive healthcare dashboard built using Power BI to analyze hospital performance, patient behavior, and financial metrics. The goal is to provide actionable insights into billing, patient distribution, and outstanding debts.
Objectives
•	Analyze hospital billing performance
•	Track patient distribution across hospitals
•	Identify outstanding debts owed by patients
•	Compare insurance provider contributions
•	Evaluate operational metrics like length of stay
 Dashboard Features
•	Key Performance Indicators (KPIs)
•	Average Billing Amount
•	Average Length of Stay
•	Outstanding Debt
Visual Insights
•	Top 5 Hospitals by Total Billing
•	Top 10 Hospitals by Number of Patients
•	Average Billing by Insurance Provider
•	Patient Distribution by Admission Type
 Tools & Technologies
Power BI
Power Query (Data Cleaning & Transformation)
DAX (Data Analysis Expressions)
Data Cleaning Process (Removed duplicates, cleaned inconsistent casing in names) using Ms Excel.
Handled missing values.
Standardized data formats.
Created calculated columns (e.g., debt identification)
Segmented data based on business logic
Key Insights
•	Certain hospitals contribute significantly to total billing
•	A noticeable portion of patients have outstanding debt
•	Insurance providers vary in average billing contribution
•	Patient distribution is uneven across hospitals.
